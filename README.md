# DHC-internship-Task-1
# Iris Dataset Exploration and Visualization

 Overview
This project focuses on exploring and visualizing the Iris dataset using Python. The main objective is to understand the structure of the dataset, analyze statistical information, and create visualizations to identify patterns, relationships, and outliers in the data.

The project uses popular Python libraries such as Pandas, Matplotlib, and Seaborn for data analysis and visualization.



 Objectives
- Load and inspect the Iris dataset
- Understand dataset dimensions and features
- Generate summary statistics
- Explore relationships between variables
- Visualize feature distributions
- Detect possible outliers using box plots

 
 Technologies and Libraries Used
- Python
- Pandas
- Matplotlib
- Seaborn

---

Dataset
The Iris dataset is a well-known dataset in machine learning and data analysis. It contains measurements of iris flowers from three different species:
- Setosa
- Versicolor
- Virginica

Features
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width
- Species



Tasks Performed

### 1. Data Loading
The dataset was loaded using the Seaborn library and stored in a Pandas DataFrame.

### 2. Data Inspection
The following operations were performed:
- Displayed dataset shape
- Printed column names
- Viewed the first few rows using `.head()`
- Used `.info()` to inspect data types and null values
- Used `.describe()` for statistical summaries

### 3. Data Visualization
Several visualizations were created to better understand the dataset:

#### Scatter Plot
Used to analyze relationships between features and compare species distribution.

#### Histograms
Used to observe the distribution of numerical values.

#### Box Plots
Used to identify outliers and understand data spread.
